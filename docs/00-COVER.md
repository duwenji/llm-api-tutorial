# LLM API 実践チュートリアル - 全体像

## このチュートリアルで身につくこと

- LLM APIが生まれた**経緯**（ビジネス背景・技術的必然性）
- リクエスト/レスポンスの**仕組み**（トークン化・メッセージ形式・ストリーミング）
- ツール呼び出し・マルチモーダル・拡張思考などの**応用機能**
- OpenAI互換API・MCPなど業界の**標準化**動向
- 実際に動くチャットクライアント/エージェントを構築する実装力

## 対象読者

- プログラミングの基礎（HTTP、JSON）を理解している方
- LLM APIをこれから体系的に学びたい方
- 生成AIを使ったアプリケーション開発に携わる方

## 学習の流れ

```
STEP 1 ──→ History and Basics（4教材）
             経緯・APIの基本仕組み・トークン・認証

STEP 2 ──→ Core Mechanics（3教材）
             メッセージとロール・ストリーミング・ツール呼び出し

STEP 3 ──→ Advanced Features（4教材）
             マルチモーダル・拡張思考・プロンプトキャッシュ・構造化出力

STEP 4 ──→ Standardization（3教材）
             OpenAI互換API・MCP・エラー/バージョニング標準
              ※ 業界全体の標準化状況の理解

STEP 5 ──→ Real World Examples（3教材）
             チャットクライアント・エージェント・コスト最適化の構築演習
```

## 前提知識

| 知識 | 要否 |
|------|------|
| HTTP/JSONの基本 | ✅ 必須 |
| コマンドライン操作 | ✅ 必須 |
| Python基本文法 | 🔶 推奨 |
| 機械学習の数学的背景 | ❌ 不要 |

## 本チュートリアルの特徴

1. 「良い例」と「悪い例」の対比で理解が深まる
2. 全コードがコピペ可能（bash/Python）で即座に動作確認できる
3. Claude APIを主軸に、OpenAI互換APIとの共通点・相違点も扱う
4. モデルIDやパラメータは執筆時点の仕様であることを明記する
