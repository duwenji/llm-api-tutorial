# 02. Core Mechanics - APIの仕組み

このカテゴリでは、LLM APIを実際に使いこなすための中核的な仕組みを学びます。
メッセージ設計・ストリーミング・ツール呼び出しは、以降の応用機能すべての土台です。

## 学習目標

- role（system/user/assistant）ごとの役割を理解する
- ストリーミング応答の受信方法と利点を理解する
- ツール呼び出し（Function Calling）の往復フローを実装できる

## 教材一覧

|---|------|------|
| 01 | [メッセージとロール](01-messages-and-roles.md) | system/user/assistantの使い分け |
| 02 | [ストリーミング応答](02-streaming.md) | SSEによる逐次受信とUI設計 |
| 03 | [ツール呼び出し](03-tool-use.md) | Function Calling往復フロー |

## 学習の進め方

01 → 03 の順に進めることを推奨します。
各教材の「理解度チェック」で習得状況を確認してください。
