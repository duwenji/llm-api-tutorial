# 05. Real World Examples - 実践演習

このカテゴリでは、これまで学んだ仕組みを組み合わせて、
実際に動くクライアント・エージェントを構築します。

## 学習目標

- 会話履歴を保持するシンプルなチャットクライアントを構築できる
- ツール呼び出しを使った自律的なエージェントを構築できる
- コストとプロバイダ選定の観点で設計判断ができる

## 教材一覧

| # | 教材 | 内容 |
|---|------|------|
| 01 | [シンプルなチャットクライアント](01-simple-chat-client.md) | 会話履歴管理とストリーミング表示 |
| 02 | [ツール呼び出しエージェント](02-tool-calling-agent.md) | 自律的なタスク遂行ループ |
| 03 | [コストとプロバイダ比較](03-cost-and-provider-comparison.md) | コスト最適化と使い分け戦略 |

## 学習の進め方

01 → 03 の順に進めることを推奨します。
各教材の「理解度チェック」で習得状況を確認してください。
