# 03. Advanced Features - 応用機能

このカテゴリでは、基本の仕組みを踏まえた応用機能を学びます。
マルチモーダル・拡張思考・キャッシュ・構造化出力は、実務での品質とコストを大きく左右します。

## 学習目標

- 画像・PDFなどテキスト以外の入力を扱えるようになる
- 拡張思考（Extended Thinking）の役割と使いどころを理解する
- プロンプトキャッシュでコストを削減する設計ができる
- 構造化出力でJSON形式を厳密に強制できる

各教材の「理解度チェック」で習得状況を確認してください。
