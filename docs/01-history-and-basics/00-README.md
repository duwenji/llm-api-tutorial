# 01. History and Basics - 経緯と基礎

このカテゴリでは、LLM APIが生まれた経緯と、APIとしての基本的な仕組みを学びます。
以降のカテゴリで扱う応用機能・標準化を理解するための土台をここで固めます。

## 学習目標

- LLM APIが登場した背景と進化の流れを説明できる
- HTTP/JSONベースのリクエスト/レスポンスの基本構造を理解する
- トークン化とコンテキストウィンドウの仕組みを把握する
- APIキー認証と主要な認証方式の違いを知る
